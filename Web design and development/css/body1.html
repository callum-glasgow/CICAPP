Size Example (with Margin and Padding)

By setting the width, padding, and margin properties, we can control the size and positioning of elements on the page. In this case:

  • The body width is set to 400px, and it is centered horizontally using margin: 0 auto.
  • The red block has a width of 200px and padding of 50px, which adds inner spacing.
  • A margin of 100px is applied, creating space outside the red block.
